# BrainstormBuddy — гид по окну настроек

> Обновлено: 2026-07-04, соответствует сборке ветки `feature/chatlog-restore`.
> Парный документ: `USER_GUIDE.md` (главный экран). Код: `SettingsWindow.xaml(.cs)`,
> `SettingsWindow.LocalStt.cs`, вью-модель `BrainstormBuddy.Core/Config/SettingsViewModel.cs`.

Окно 720×600 (мин. 560×460): сайдбар слева (активная вкладка — зелёная полоса + подсветка),
контент справа, футер снизу. Открытие: шестерёнка в оверлее, `Ctrl+Shift+S`, трей,
автоматически при пустом API-ключе.

## Футер (общий для всех вкладок)

| Кнопка | Что делает |
|---|---|
| Открыть папку | Проводник в `%AppData%\BrainstormBuddy` (конфиг, логи, экспорты) |
| **Сохранить** (зелёная) | Пишет конфиг на диск + применяет на лету (прозрачность, тема, аудио-параметры). Окно НЕ закрывается — можно крутить дальше |
| Закрыть | Закрывает окно БЕЗ сохранения на диск (изменения в памяти живут до выхода из приложения — при выходе конфиг сохраняется целиком) |

## Вкладка 1 · API

| Поле | Тип | Описание |
|---|---|---|
| Пресет | комбо | 7 вариантов: Groq · OpenAI · OpenRouter (gpt-oss-120b) · OpenRouter (free) · NVIDIA NIM (llama-3.3-70b) · LocalAI · Custom. Выбор заполняет BaseUrl/модели разом. `Custom` — ручной ввод. ⚠ Сейчас значения LLM принудительно перекрываются кодом на локальный Ollama (см. «Подводные камни») |
| BaseUrl | текст | Chat-эндпоинт (OpenAI-совместимый, без `/chat/completions` на конце). Текущий: `http://127.0.0.1:11434/v1` (Ollama) |
| API Key + кнопка-глаз | текст | Ключ провайдера; глаз переключает видимость (текст ↔ точки). Для локальных Ollama/GigaAM не нужен — пустой |
| Chat Model | текст | Имя модели чата. Текущая: `qwen2.5vl:3b` |
| STT BaseUrl (опционально) | текст | Отдельный эндпоинт распознавания; пусто = использовать BaseUrl. Текущий: `http://localhost:8765/v1` (Docker GigaAM) |
| STT Model | текст | Модель распознавания. Текущая: `gigaam-v2` |
| RateLimit между LLM-запросами (сек) | число | Минимальная пауза между запросами к LLM (защита от 429 у облачных). Дефолт 15; для локального Ollama можно 0 |
| Проверить подключение | кнопка | GET `/models` на BaseUrl; результат справа: зелёное «Подключение успешно» / красная ошибка |

## Вкладка 2 · LLM (системные промпты)

Слева — список пресетов промптов (7 дефолтных: Собеседование: шпаргалка · Совещание:
аналитика · Мозговой штурм · Протокол встречи · Продажи: работа с возражениями ·
Перевод RU-EN · Технический консультант). Выбранный пресет = активная роль LLM
(она же в селекторе тайтлбара оверлея).

| Элемент | Описание |
|---|---|
| Имя пресета | текст, редактирует имя выбранного |
| Редактор промпта | большое mono-поле, правки применяются к выбранному пресету |
| Сохранить как новый | копия текущего текста под новым именем |
| Удалить пресет | удаляет выбранный |
| + Новый пустой | создаёт пустой пресет |
| Объединять [Динамик] и [Микрофон] в один запрос | чекбокс: склеивать реплики обеих сторон в одно сообщение LLM |
| Размер истории чата | слайдер 2–20 сообщений (дефолт 6): сколько прошлых реплик LLM видит как контекст |

## Вкладка 3 · Аудио

| Элемент | Диапазон / варианты | Описание |
|---|---|---|
| Пресет аудио | Стандарт · Быстрая речь · Медленная/Вдумчивая | Заполняет 4 слайдера ниже разом (Стандарт: 1800/1000/400/500 мс) |
| Микрофон (вход) | комбо устройств | «По умолчанию» или конкретное устройство записи |
| Динамик (loopback) | комбо устройств | Источник системного звука (голоса собеседников) |
| Порог тишины | 500–2500 мс | Пауза в речи, после которой реплика уходит в STT (= слайдер ТИШИНА в оверлее, там в секундах) |
| Мин. длительность речи | 200–1500 мс | Короче — считается щелчком/шумом и отбрасывается |
| Pre-roll | 100–500 мс | Сколько звука ДО начала речи прихватить (чтобы не резало первое слово) |
| Post-roll | 100–500 мс | Сколько звука ПОСЛЕ конца речи прихватить |
| RMS-порог | 0.005–0.1 | Чувствительность детектора голоса. Ниже — ловит шёпот и шум, выше — только громкую речь |
| **Live RMS** | индикатор | Живая громкость ~7 раз/сек: точка зелёная = звук выше порога (детектор сработает), серая = тишина. Главный инструмент калибровки порога |
| Только микрофон | чекбокс | Отключить захват системного звука |
| Захватывать микрофон | чекбокс | Вкл. второй канал (своя речь) отдельным VAD-пайплайном — реплики получают метку [Микрофон] |
| Режим VAD | 0–3 | 0 Качество (тихая речь) · 1 LowBitrate · 2 Aggressive (совещания) · 3 VeryAggressive (шумный фон; RMS+ZCR двойной фильтр) |
| Отладочные логи аудио | чекбокс | RMS в лог каждую секунду (для диагностики) |

## Вкладка 4 · Локальный STT (Docker)

Управление контейнером GigaAM (русское распознавание, полностью локально).
Требуется Docker Desktop; первый запуск качает модель ~450 МБ (до 5 минут).

| Элемент | Описание |
|---|---|
| Использовать локальный STT | чекбокс: переключает STT приложения на `localhost:8765` + `gigaam-v2` (обратно — восстанавливает прежние адреса) |
| Статус (точка + текст) | Остановлен / Запускается… / Сборка образа… / **Работает — адрес** (зелёная) / Ошибка (красная) |
| Запустить (зелёная) | Стартует существующий контейнер, а если его нет — собирает образ из `LocalSttBridge/` и создаёт |
| Остановить | `docker stop` — контейнер на паузе, данные целы |
| Удалить | `docker rm` — контейнер уничтожается (кэш модели живёт в named volume и переживает удаление) |
| Пинг | Проверка `/models` с замером времени ответа |
| Метрики (mono) | CPU / RAM / ping / uptime контейнера, обновляются каждые ~3 с |
| Лог сборки/запуска | Стрим docker-вывода в реальном времени (автопрокрутка) |

## Подводные камни

1. **Force-override**: при старте `App.xaml.cs` принудительно ставит LLM = локальный Ollama
   (`127.0.0.1:11434/v1`, `qwen2.5vl:3b`), а STT — по чекбоксу «Использовать локальный STT».
   Поэтому смена BaseUrl/Chat Model на вкладке API переживёт сессию, но после перезапуска
   вернётся на локалку. Это осознанно (нет бюджета на облако); чтобы изменить — править force-set в коде.
2. Смена активного пресета промпта (здесь или в оверлее) **сбрасывает диалог** (контекст + токены).
3. «Сохранить» не закрывает окно — это фича (BUG #25), закрытие отдельной кнопкой.
4. Порог тишины здесь в мс, а слайдер ТИШИНА в оверлее — в секундах; это одна и та же настройка.
5. Ключ API хранится открытым текстом в `%AppData%\BrainstormBuddy\config.json`.
